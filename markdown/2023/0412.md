## 問1 : 九九の合計

### 私が入力した文章

```
ワンライナーのBashスクリプトで九九の掛け算（1×1～9×9まで）の答えの合計を求めてください。
```

### ChatGPTが書いたBashスクリプト

```
sum=0; for i in {1..9}; do for j in {1..9}; do sum=$((sum + i * j)); done; done; echo $sum
```

### 実行結果

```
2025
```

### 私の感想

この調子でもう一問、解いていただきましょう。
